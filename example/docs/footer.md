# Footer (底部栏)
##### 默认固定在底部
###### 常见组件
---
## 示例
### 简单使用
默认背景色为白色，固定在底部
```html
<kw-footer bg-color="success">绿色背景</kw-footer><br/>
<kw-footer color="danger">红色文字</kw-footer>
```
# API
## Props
名称|描述|类型|可选值|默认值
---|---|---|---|---
color|参见Bar组件
bg-color|参见Bar组件
fixed|是否固定在底部|boolean|true,false|true
height|参见Bar组件
bottom|参见Bar组件
